üîπ Objective:
The objective of this project is to design and implement a hotel network infrastructure using Cisco Packet Tracer. The key components include:
- Implementation of multiple VLANs to separate departments.
- Wireless Access Points to enable mobility for laptops and mobile devices.
- IP Phones (VoIP) for efficient voice communication.
- DHCP configuration for automatic IP address assignment.
- Routers and switches to ensure full network interconnectivity.
- A hierarchical topology that supports future scalability and maintenance.

üîó Connectivity
- Each VLAN is connected via dedicated switches and linked to a central router.
- Routers (Cisco 2811) are used to connect different departments and floors.
- DHCP is configured to automatically assign IP addresses to devices.
- Wireless access is available in the Admin Office, Reception Area, and Second Floor.
- VoIP phones are installed in the IT Department, Second Floor, and Third Floor for internal communication.

‚úÖ Key Technologies Used
